# Store - Spring Boot Learning Project

A personal learning project to gain knowledge in modern app development with Java Spring Boot.

## Learning Materials

### [Spring Boot Fundamentals](https://codewithmosh.com/p/spring-boot-fundamentals)

**Core concepts covered:**

- Dependency injection and Spring Boot object management
- Database connectivity with Spring Data repositories
- Application configuration using properties and YAML files
- Hibernate entity tracking and transaction management
- Fetching strategies: eager and lazy loading
- Custom queries with @Query, JPQL, and native SQL
- Query optimization with projections, specifications, and criteria API
- Sorting, pagination, and handling large datasets
- Code simplification with Lombok and productivity tools

Full Curriculum: <https://codewithmosh.com/p/spring-boot-fundamentals>

### [Spring Boot: Building APIs](https://codewithmosh.com/p/spring-boot-building-apis)

**Practical skills covered:**

- Building RESTful APIs
- Request validation and error handling
- Security with JWT authentication and role-based access control
- Clean architecture and project structure
- Shopping cart and checkout functionality
- Payment integration with Stripe Checkout
- Environment-specific configuration with Spring profiles
- Cloud deployment
- Production-ready best practices

Full Curriculum: <https://codewithmosh.com/p/spring-boot-building-apis>
